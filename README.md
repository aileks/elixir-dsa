## Getting Started

1. Clone the repository
```bash
git clone https://github.com/aileks/elixir-dsa.git
```

2. Navigate to directory
```bash
cd elixir-dsa
```

3. Run tests

```bash
mix test
```

4. Edit the files in `lib` to pass tests!
